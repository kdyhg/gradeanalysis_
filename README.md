# 담임 성적 메시지 스튜디오

나이스 성적통지표 엑셀 파일을 업로드해 학생별 과목 성적을 분석하고, 가정으로 보낼 개별/단체 메시지 초안을 생성하는 Next.js 앱입니다.

## 주요 기능

- 나이스 성적통지표 엑셀의 학생별 반복 블록 자동 분리
- 과목별 점수, 과목 평균 대비 차이, 석차, 동석차, 수강자수 분석
- 2022 개정 교육과정 5등급제 기준 추정 등급 산출
- 학생별 강점/점검 과목과 학급 과목별 흐름 요약
- Gemini API 또는 OpenAI API를 이용한 학부모 메시지 초안 생성
- API 키가 없을 때도 로컬 규칙 기반 초안 생성

## 5등급제 기준

앱은 석차에 동석차수가 있는 경우 `석차 + (동석차수 - 1) / 2` 방식의 중간석차를 사용해 백분율을 계산합니다.

| 등급 | 누적 비율 |
| --- | --- |
| 1등급 | 상위 10% 이내 |
| 2등급 | 상위 34% 이내 |
| 3등급 | 상위 66% 이내 |
| 4등급 | 상위 90% 이내 |
| 5등급 | 상위 90% 초과 |

학교별 학업성적관리규정과 동점자 처리 지침이 다를 수 있으므로, 앱의 등급은 담임 상담용 추정값으로 사용하세요.

## 실행

```bash
npm install
npm run dev
```

`.env.local`에 다음 값을 넣으면 Gemini API로 문안을 생성합니다.

```bash
GEMINI_API_KEY=your-gemini-api-key
GEMINI_MODEL=gemini-2.5-flash-lite
```

OpenAI를 보조 공급자로 쓰려면 다음 값도 사용할 수 있습니다.

```bash
OPENAI_API_KEY=sk-your-key
OPENAI_MODEL=gpt-5.4-mini
```

## 개인정보

- 엑셀 파싱과 기본 분석은 브라우저에서 처리됩니다.
- AI 문안 생성 버튼을 누를 때 선택된 학생 요약 또는 학급 요약만 서버 API로 전송됩니다.
- 학생 성적 파일과 `.env.local`은 Git에 포함하지 않도록 제외했습니다.
